---
description: K8s 지식 베이스에서 키워드를 검색합니다. 사용법: /search-kb <키워드>
---

`$ARGUMENTS` 키워드로 지식 베이스를 검색합니다.

## 검색 범위
- `components/`, `objects/`, `network/`, `security/`, `deep-dive/` 전체 마크다운 파일

## 검색 방법
1. 파일명에서 키워드 매칭
2. 파일 내용 중 헤더(#) 라인에서 키워드 매칭
3. 관련 개념이 언급된 섹션 요약

## 출력 형식

```
## 검색 결과: "{키워드}"

### 관련 문서
| 파일 | 관련 섹션 | 연관도 |
|------|---------|-------|
| {파일명} | {섹션명} | High/Medium/Low |

### 추천 문서
{가장 관련성 높은 문서 경로와 핵심 내용 요약}
```
